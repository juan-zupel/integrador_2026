## Mi Proyecto Web

## Descripción
Esta página web está destinada a dar diversas guías cortas sobre formas poco comunes de jugar al videojuego "League of Legends", este proceso se realiza mediante guias/tutoriales cortos distribuidos a lo largo de toda la página, los cuales pueden ocultarse y mostrarse rapidamente a elección del usuario.
Adicionalmente se brinda a los usuarios la capacidad de solicitar un plan de coacheo (el coacheo
conciste en que una persona experimentada te de consejos o te guíe, ya sea durante o despues de cada partida).

## Contenido
    breve explicación del juego
    espacio publicitario
    guias separadas por personajes y posciciones, incluye tambien:
        imagenes
        cuadros comparativos
        observaciones
    formulario de inscripción para coacheo
    informacion de contacto

## Tecnologías usadas
    HTML: para la estructura
    CSS: para el estilo, lo que incluye:
        Implementacion de fuentes mediante "links"
        Organizacion responsive mediante "@keyframes"
    JavaScript: para la implementación de dinamismo dentro del sitio WEB, incluye:
        Implementación de funciones encargadas de manejar diversas funcionalidades
        Manejo/modificación de los elementos que conforman la interfáz gráfica
        Implementación de pequeñas animaciones/ayudas visuales

## Mejoras visuales
La página se estructura de la siguiente manera:
"--------título----------"
"publicidad-----contenido"
"------acesoraminto------"
"-----pie-de-pagina------"
Siendo el "contenido principa" (los tutoriales) lo que mas espacio abarca dejando a los otros elementos como simples 
ayudas visuales que dornan un poco la pagina teniendo tambien su parte de contenido.
Al achicar la pantalla, la estructura general no cambia mucho y mantiene la coherencia en todo momento (o eso intenta).

## Funcionalidades Implementadas
    Boton encargado de ocultar y mostrar espacio publicitario.
    Botones encargados del despliegue/repliegue de las guias/tutoriales.
    Sistema de filtrado de tutoriales por especificación del usuario.
    Sistemas de control de errores y validación de información en sección de formulario.
